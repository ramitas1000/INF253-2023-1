El archivo makefile tiene que estar en la misma carpeta que la carpeta Tarea3 (NO dentro de ella)
Para compilar los archivos, todos deben estar en la carpeta Tarea3 y se debe usar el comando make compile
Para ejecutar el programa se debe usar make run
Para borrar los archivos creados por el compilador se debe usar make clean

Los inputs disponibles se muestran al ingresar el nombre del jugador, el ancho y el alto.
El mapa no es ciclico, tiene bordes
