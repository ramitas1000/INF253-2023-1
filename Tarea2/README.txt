Para compilar los archivos, todos debenestar en una misma carpeta y se debe usar make compile
Para correr el programa se debe usar make run
Para correr con valgrind se debe usar make runvalgrind
El programa no libera casi nada de memoria

Las funciones estan comentadas en arbol.c
Las estructuras estan comentadas en arbol.h
El main tiene breves comentarios de su funcionamiento

Los comandos que puede leer el programa son:
	touch
	write
	cat
	ls
	mkdir
	break
